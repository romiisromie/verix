# 🚀 Развертывание Verix на публичном хостинге

## Способ 1: Netlify Drop (самый простой)

1. **Откройте** [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. **Перетащите** файл `index.html` в окно загрузки
3. **Готово!** Получите публичную ссылку вида `https://random-name.netlify.app`

## Способ 2: GitHub Pages

1. **Создайте аккаунт** на [GitHub](https://github.com) если его нет
2. **Создайте новый репозиторий** с названием `verix-app`
3. **Загрузите файл** `index.html` в репозиторий
4. **Включите GitHub Pages** в Settings → Pages
5. **Готово!** Ссылка: `https://username.github.io/verix-app`

## Способ 3: Vercel

1. **Откройте** [https://vercel.com](https://vercel.com)
2. **Войдите через GitHub**
3. **Нажмите "New Project"**
4. **Загрузите** файл `index.html`
5. **Готово!** Получите ссылку Vercel
