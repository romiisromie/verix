## 🚀 Быстрый старт

### Локальная разработка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/romiisromie/verix.git
   cd verix
   ```

2. **Запустите локальный сервер**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

3. **Откройте в браузере**
   ```
   http://localhost:8000
   ```
